# COVID 19 - Belgium
Statistics &amp; model on covid spreading in Belgium.

Other countries DB will be added soon. 
Also, it will include a model based on https://www.nature.com/articles/s41421-020-0148-0.pdf

Wang, H., Wang, Z., Dong, Y. et al. Phase-adjusted estimation of the number of Coronavirus Disease 2019 cases in Wuhan, China. Cell Discov 6, 10 (2020). https://doi.org/10.1038/s41421-020-0148-0

Data are collected from Sciensano when running the software. So data are always the last available. 

> \>\> python GUI.py

The software has a built-in Graphical User Interface (GUI).

Tested with Python 3.8.
